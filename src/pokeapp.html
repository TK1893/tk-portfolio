POKEhome - A JavaScript WEb App

PokeHome App

The goal of the PokeHome project is to develop a small web application using HTML, CSS, and JavaScript. The app will utilize an external API to fetch data. It allows users to view a list of data items and access detailed information on individual entries.

Project Context

This project focuses on building a complete JavaScript application. The main focus is on the technical functionality. The app should also be aesthetically pleasing and user-friendly.

Key Features

  • Data Loading

    The application fetches data from an external API, such as the Pokémon API.

  • Data Display

    The app presents a list of items that becomes available once the page loads.

  • Detail View

    Users can interact with the list (e.g., by clicking) to view detailed information about a selected item.

  • Styling

    CSS is used to ensure the user interface is visually appealing. A UI framework/library such as Bootstrap can optionally be used for styling.

  • Complex UI Elements

    The app includes at least one complex user interface pattern, such as a modal for displaying details or touch interactions.

Technical Requirements

  • The app must function across various web browsers (Chrome, Firefox, Safari, Edge, Internet Explorer 11).
  • JavaScript code should adhere to ESLint guidelines.
  • The application should be deployed on a publicly accessible platform like GitHub Pages.
