KraftFlix - Movies

KraftFlix App

Kraftflix is a full-stack web application that allows movie enthusiasts to browse a movie database for information about different movies, directors, and genres. Users can create profiles, save their favorite movies, and manage their personal data. The application is built using the MERN stack (MongoDB, Express, React and Node.js). The backend provides a REST API and MongoDB database. The frontend offers a responsive, user-friendly interface.

The Kraftflix project is showcasing skills in both server-side and client-side development. The backend, built using Node.js, Express, and MongoDB, will handle API requests, user authentication, and data storage. The frontend, built using React, will provide users with an interactive, responsive experience, allowing them to explore and manage movie information and their personal favorites list.

Backend

The backend of Kraftflix is focused on creating the server-side components of the application. This includes setting up a REST API that interfaces with a MongoDB database to store movie data, user information, and perform CRUD (Create, Read, Update, Delete) operations. The backend will handle data requests made by the frontend and ensure user data security and authentication.

Key Features

  • Movie Information

    Return a list of all movies

    Retrieve detailed data on a single movie (title, description, genre, director, image)

    Retrieve information about genres (e.g., “Thriller”) and directors (bio, birth year, etc.)

  • User Management

    Allow new users to register.

    Allow users to update their profile (username, password, email, date of birth).

    Users can add or remove movies from their favorite list.

    Enable users to deregister their account.

Technical Requirements

  • Node.js, Express, and MongoDB are used for server-side development.
  • RESTful architecture with well-defined endpoints.
  • Middleware modules (e.g., body-parser, morgan) are implemented.
  • Authentication using HTTP and JWT tokens.
  • Data validation and security protocols are mandatory.
  • The backend must be deployed on Heroku and the source code hosted on GitHub.

Frontend

The frontend of Kraftflix will be a single-page application (SPA) built using React. It will interface with the backend API to display movie data, manage user sessions, and enable users to modify their profiles and movie preferences. The frontend will provide a polished, responsive user experience that can be accessed on any device.

Views & Features

  • Main View

    Display a list of all movies with search and filter functionality.

    Ability to select a movie for more detailed information.

    Provide navigation to the profile view and allow users to log out.

  • Single Movie View

    Show detailed data about a selected movie (description, genre, director, image).

    Allow users to add or remove the movie from their favorites list.

  • Login View

    Users can log in with their username and password.

  • Signup View

    New users can register by providing a username, password, email, and date of birth.

  • Profile View

    Show user details and allow them to update their profile information.

    Display the user’s favorite movies and provide the option to remove movies from the list or deregister the account.

Technical Requirements

  • React is used with ES2015+ for modern JavaScript development.
  • Parcel is employed as the build tool.
  • Bootstrap ensures a responsive UI.
  • State management can optionally use React Redux for features like movie filtering.
  • The UI will be deployed online, making it accessible from any device.
